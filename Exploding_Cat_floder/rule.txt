開局丟牌
在使用功能前如果有人nope
停止效果(除了爆炸貓 拆除卡)以外都可執行
若沒有 執行效果
1.attack-結束回合後不須抽牌 下回合的人需要執行2次回合
2.skip-無須抽卡直接跳過(不能再丟其他卡) 如果被攻擊卡執行效果 要2張才能
完全消除 一張的話只能正常執行回合
3.favor-強制指定一位玩家給一張牌(給予方決定)
4.shuffle-洗牌
5.see the future-能看見牌庫前3張
6.nope-停止 上述
7.defuse-抽到爆炸貓丟出
8.exploding cat-你輸了
9.特殊連擊(可以被nope)
抽一張牌 結束回合

細節流程
出牌
if 任何人有nope:
    詢問是否要出nope
else:
    if card == skip:
        直接跳過
    else:
        執行效果
詢問是否要再出牌
if yes:
    執行上述
else:
    抽一張牌
    if card==exploding cat:
        輸了
    else:
        結束回合
